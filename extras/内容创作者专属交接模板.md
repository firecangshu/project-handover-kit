| 08-决策记录.md | ⭐⭐⭐ | 为什么选这个方向、哪些选题验证过 |

### 使用技巧

1. **风格描述要具体**：不要只说"风格要高级"，要说"用 XX 博主的那种风格，短句多，有画面感，不用成语"
2. **给样本比描述管用**：把你觉得写得好的 3-5 篇范文放进记忆包，AI 模仿样本比听描述准 10 倍
3. **禁忌雷区写清楚**：什么不能写、什么词不能用、什么话题不能碰，越详细越好
4. **数据反馈要记录**：哪篇爆了、为什么爆、哪篇凉了、为什么凉，这些是最宝贵的经验

## 常用 Prompt 模板

### 让 AI 接手写文案

```
我给你一份账号交接包，你先完整读一遍，然后告诉我：
1. 这个账号是做什么的
2. 目标受众是谁
3. 内容风格是什么样的
4. 有哪些禁忌雷区

确认你理解了，我们再开始写内容。

【附上 IRY 记忆包】
```

### 写小红书爆款笔记

```
按照我们账号的风格，写一篇小红书笔记，主题是：【主题】

要求：
1. 标题要有钩子，让人想点进来
2. 开头 3 秒抓住注意力
3. 内容有干货，不是空喊口号
4. 结尾有互动引导
5. 配 5-8 个相关标签

参考我们之前的爆款风格。

【附上记忆包 + 爆款样本】
```

### 选题 brainstorm

```
我们账号的定位是：【定位】
目标受众是：【受众】

帮我想 20 个选题方向，按照以下分类：
1. 干货类（5 个）
2. 情感共鸣类（5 个）
3. 争议/话题类（5 个）
4. 互动类（5 个）

每个选题给一个标题思路和一句话核心观点。
```

- [ ] 账号定位和目标受众

- [ ] 排期计划
- [ ] 平台规则红线
