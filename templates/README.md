## 怎么用？

### 方法一：作为 IRY 生成后的补充参考
1. 先用 IRY 自动扫描生成记忆包
2. 再对照对应场景的模板，检查有没有遗漏的关键点
3. 手动补充模板里有但 IRY 没扫到的信息

### 方法二：直接基于模板填写
1. 复制对应场景的模板文件
2. 按照模板的结构和提示，手动填写你的项目信息
3. 填写完成后，就可以作为交接包使用了

### 方法三：混合使用
- IRY 自动扫描 + 模板人工补充，效率最高，信息也最完整
